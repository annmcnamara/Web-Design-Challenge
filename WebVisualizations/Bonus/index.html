This Project Analysed National Park Visitor Data


Park data was retrieved using the National Park APIs from National Park Data and National Park Visitor Data
Weather data was retrieved using the Dark Sky API.
Gas Price data was retrieved using the EIA (US Energy Informaton Association) API.

(JSON) Data was retrieved from the APIs above and loaded intto dataframes using using python and pandas.

The interacive plots linked on this page were created using plotly.
